# PTP client and server component for ESP-IDF

This component was originally ported from [Nuttx PTP Daemon](https://github.com/apache/nuttx-apps/tree/master/netutils/ptpd) to ESP-IDF as an example implementation by Ondrej Kosta at Espressif. Since then it has been enhanced to support gPTP (802.1AS) by Scramble Tools, and it is maintained by Scramble and registered at the ESP Component Registry (https://components.espressif.com).
